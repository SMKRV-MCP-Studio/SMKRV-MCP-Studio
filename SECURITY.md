# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
| < latest | No       |

Only the latest Docker image tag receives security updates.

## Reporting a Vulnerability

**Do NOT open a public issue for security vulnerabilities.**

Email **[ms@smcps.net](mailto:ms@smcps.net)** with:

- Description of the vulnerability
- Steps to reproduce
- Affected version / Docker tag
- Impact assessment (if known)

You will receive a response within 72 hours. Confirmed vulnerabilities will be patched in the next release and credited (unless you prefer to remain anonymous).

## Scope

This policy covers the Docker image `smkrv/smkrv-mcp-studio` and the web application it runs. It does not cover:

- User-written SQL queries or MCP tools
- External databases connected by users
- Third-party infrastructure
